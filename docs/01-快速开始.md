# 🚀 5 分钟快速开始

欢迎！本指南帮助您在 5 分钟内完成首次刷机。

## 推荐路径：Web Studio 🌐

**nRF5-AirTag-Toolkit** 提供了业界最优雅的 Web 控制台，无需记忆任何命令。

### 第 1 步：启动 Web Studio

```bash
python3 nrf5_airtag_web.py
```

> 💡 默认地址：`http://127.0.0.1:5001`

### 第 2 步：连接硬件

将调试器（J-Link / ST-Link / DAPLink）通过 SWD 接口连接到您的nRF芯片：

```
调试器           nRF 芯片
------          --------
SWDIO    -->    SWDIO
SWCLK    -->    SWCLK
GND      -->    GND
3.3V     -->    VCC (可选，如果芯片无独立供电)
```

### 第 3 步：开始刷机

1. 打开浏览器访问 Web Studio
2. **自动检测**：系统会识别您的调试器和芯片型号
3. 配置设备名称（如 `MSF001`）
4. 选择密钥模式：
   - **Dynamic（推荐）**：无限动态密钥，追踪效果最佳
   - **Static**：固定密钥
5. 点击 **"START FLASHING"**
6. 等待进度条完成
7. 下载生成的密钥包（`.zip`），导入到您的 Find My 应用

## 高级路径：命令行工具 ⚡

如果您需要批量刷机或无 GUI 环境，使用命令行工具：

```bash
./nrf5_airtag_flash.sh
```

该脚本会交互式引导您完成：

- 芯片选择（nRF51822 / 52832 / 52810）
- 密钥模式选择
- 调试器选择
- 批量循环刷写

详见：[命令行刷写工具完全指南](./04-命令行刷写工具.md)

# nRF52810 & heystack-nrf5x 完整文档体系

## 📚 文档概览

本文档体系提供了从零开始到精通 nRF52810 固件开发的完整指南，适合不同层次的开发者使用。

## 📖 文档结构

### 🚀 入门必读 (按顺序阅读)

#### 1. [环境安装指南](01-环境安装指南.md)

- **适用人群**: 初次搭建开发环境
- **主要内容**:
  - macOS 开发工具安装
  - Homebrew、ARM GCC、OpenOCD 配置
  - 环境验证脚本
- **预计时间**: 30-60分钟

#### 2. [项目配置手册](02-项目配置手册.md)

- **适用人群**: 环境安装完成后
- **主要内容**:
  - nRF5 SDK 下载配置
  - 项目路径设置
  - 工具链配置
- **预计时间**: 20-30分钟

#### 3. [固件编译指南](03-固件编译指南.md)

- **适用人群**: 需要理解编译过程
- **主要内容**:
  - SoftDevice 合并原理
  - 编译参数详解
  - 密钥补丁机制
- **预计时间**: 30-45分钟

#### 4. [硬件连接与刷写](04-硬件连接与刷写.md)

- **适用人群**: 准备刷写固件 (ST-Link V2)
- **主要内容**:
  - ST-Link V2 连接方法
  - 刷写命令和流程
  - 故障排除指南
- **预计时间**: 15-30分钟

#### 5. [J-Link 刷写指南](09-JLink刷写指南.md)

- **适用人群**: 使用 J-Link 调试器
- **主要内容**:
  - J-Link 连接与供电
  - nRF52810/nRF52832 自动化脚本使用
  - **救砖指南 (Recover/Manual Unlock)**
- **预计时间**: 15-30分钟

#### 5.1 [nRF52832 刷机保姆级教程](10-nRF52832刷机保姆级教程.md) ⭐ **小白推荐**

- **适用人群**: 零基础新手 (nRF52832)
- **主要内容**:
  - 从硬件连线到密钥生成的一站式教程
  - 图文并茂，手把手教学
- **预计时间**: 10分钟

### 📋 日常参考

#### 5. [快速参考手册](05-快速参考手册.md) ⭐ **最常用**

- **适用人群**: 日常开发使用
- **主要内容**:
  - 常用命令速查表
  - 参数配置对照
  - 故障排除清单
- **使用场景**: 忘记命令时快速查找

### 🔬 深入理解

#### 6. [技术原理解析](06-技术原理解析.md)

- **适用人群**: 希望深入了解技术细节
- **主要内容**:
  - nRF52810 芯片架构
  - AirTag 协议原理
  - 功耗优化策略
- **预计时间**: 60-90分钟

#### 7. [项目备份与部署指南](07-项目备份与部署指南.md)

- **适用人群**: 需要备份或部署项目
- **主要内容**:
  - 项目备份方法
  - 跨环境部署
  - 完整性验证
- **预计时间**: 20-30分钟

#### 8. [项目备份成功指南](08-项目备份成功指南.md)

- **适用人群**: 查看备份包详情
- **主要内容**:
  - 备份包结构说明
  - 使用方法展示
  - 特色功能介绍
- **预计时间**: 10-15分钟

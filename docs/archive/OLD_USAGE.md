# 使用说明

## 快速开始
```bash
# 1. 检查环境
./scripts/one_click_verify.sh

# 2. 如果环境不完整，运行安装脚本
./scripts/one_click_install.sh

# 3. 连接硬件后刷写固件
./scripts/compile_and_flash_2s.sh
```

- 查看 docs/README-文档导航.md 了解完整文档结构
- 新手请按顺序阅读 01-04 号文档
- 日常使用参考 05-快速参考手册.md
